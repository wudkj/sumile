### 项目介绍
sumile是由sumile集合各种方法或者第三方库开发的一个程序示例软件，方便在以后使用的时候进行查找。

今后内容会逐渐增加。

本程序除点按屏幕外，长按在不同的页面中会有展示代码，复制内容的功能，方便使用。
